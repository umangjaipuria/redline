Welcome to Redline

Redline is a local review surface for reviewing AI-written HTML documents. Select any text, leave a comment, and it appears in the margin beside what you marked.

How comments are stored

Comment locations are marked with lightweight inline anchors wrapped around the reviewed text, and the thread messages live in an inert application/json script tag. The document and its open review state stay together in one HTML file.

The working loop

Set your name in the top bar, select text, and click Comment. An agent can read the same state, edit the HTML, reply to your threads, and resolve threads once the requested changes are complete. Changes autosave back to the original file, and the open browser updates as the document changes underneath it.
